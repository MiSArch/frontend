# frontend
The frontend of the MiSArch reference architecture: The web application for the online shop.

## Project setup

```
# yarn
yarn
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev
```

### Compiles and minifies for production

```
# yarn
yarn build
```

### Lints and fixes files

```
# yarn
yarn lint
```
